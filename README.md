## Capstone_Design
- 프로젝트명: 동아리 관리 플랫폼

- 교내 동아리를 효율적으로 관리하고, 학생간 동아리간 의사 소통을 원활히 할 수 있는 동아리 관리 서비스 제공
- 팀명:  3박자  
- 팀원
   - 김진섭
   - 김정인
   - 박태언 
 
## 기대효과
 - 동아리 관리 플랫폼을 통해 학생과 동아리의 소통을 원할하게 함.
 - 플랫폼을 통해 지속적인 동아리 홍보 가능.
 - 동아리 회원 관리 및 다양한 정보 서비스 제공 받을 수 있음.
